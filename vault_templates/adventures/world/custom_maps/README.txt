Custom Maps Directory

This directory is for storing custom map images and map-related files for your adventures.

Supported file types:
- .png - PNG image files
- .jpg/.jpeg - JPEG image files
- .svg - Scalable Vector Graphics
- .map - Azgaar's Fantasy Map Generator files

Naming convention:
- Use descriptive names: "kingdom_map.png", "dungeon_level_1.jpg"
- Avoid spaces, use underscores: "tavern_interior.png"
- Include scale or context: "world_map_large.png", "city_district_detail.png"

Map files should be:
- High quality and readable
- Appropriately sized for their purpose
- Named consistently with your adventure structure

The system will automatically detect and load map files from this directory.
